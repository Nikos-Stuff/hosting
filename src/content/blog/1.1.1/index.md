## Tweaks for Hytale

To provide the best possible performance and ensure stable gameplay, we’ve made several improvements specifically for **Hytale**. These changes are designed to boost overall TPS and reduce potential bottlenecks.

### Performance & Optimization
- **Tweaked default mod configuration**
  - Optimized default values to reduce unnecessary overhead.
  - Improved compatibility between commonly used mods.
  - Reduced excessive background processing caused by non-critical features.

- **Tweaked how resources are allocated**
  - Improved CPU and memory allocation logic to better handle peak loads.
  - Reduced resource starvation during high player activity.
  - More efficient scaling when additional resources are available.

### Stability & Reliability
- **Fixed issues with the installation script**
  - Resolved edge cases where installations could fail or hang.
  - Improved error handling and validation during setup.
  - Ensured consistent behavior across fresh installs and updates.
